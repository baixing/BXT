### 本期演讲简介

在你的头脑里，CSS 是什么？是一团枯燥的代码，还是一堆乏味的工具？在与项目进度日复一日的缠斗中，我们是否早已遗忘了 CSS 最初带给我们的欣喜？

百姓网的 “CSS魔法” 同学尝试找寻这份遗失的感动。在他的演讲中，有真实的网页案例，有纯粹好玩的技巧，更有乍看起来完全不可能用 CSS 实现的神奇效果；他将在你的惊叹声中探索 CSS 所能达到的极致，一起重拾 CSS 的乐趣。
